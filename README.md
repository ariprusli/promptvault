markdown# PromptVault

A local prompt documentation manager built with React + Vite. Store, version, organize, and compare your AI prompts — all saved in your browser.

## Features

- **Prompt editor** — write and edit prompts with version history
- **Versioning** — create multiple versions per prompt, label them, track which model they target
- **Side-by-side diff** — compare any two versions with word-level highlighting
- **Groups** — prompts auto-group by type, with custom rename support
- **Tags & filters** — tag prompts and filter by status or tag
- **Links** — attach reference links (docs, playground, testing) per prompt or globally
- **Dark / light mode** — toggle with ☀️/🌙, preference is saved
- **localStorage persistence** — all data stays in your browser, no backend needed
